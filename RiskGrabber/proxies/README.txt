Прокси для парсера (banki.ru)
==============================

Положите в эту папку один или несколько файлов с расширением .txt.
В каждом файле — по одному прокси на строку.

Формат строки:
  http://host:port
  http://user:password@host:port
  https://host:port
  или просто host:port (подставится http://)

Строки, начинающиеся с #, и пустые строки игнорируются.
Парсер при каждом запросе случайно выбирает прокси из списка.

Пример (proxies.txt):
  http://proxy1.example.com:8080
  http://user:pass@proxy2.example.com:3128
  # 192.168.1.1:8888

Переменная окружения PROXIES_DIR задаёт другую папку (абсолютный или относительный путь).
По умолчанию используется эта папка (RiskGrabber/proxies).
